Detailed Description

Singleton coordinating display profiles, camera models, and thread-safe pipeline execution.

GazeServer is the core architectural component of godot-gaze. It manages spatial entities (Displays, Cameras, Faces, Eye Trackers) using RIDs, executes processor pipelines on a background thread, applies calibration, projects gaze rays onto displays, and defer-emits signals to Godot clients.
