Computes the pitch and yaw (in degrees) of this direction vector.

Unlike standard spherical coordinates, this computes the yaw relative to a customizable reference direction (defaulting to the forward/negative-Z direction

  • GazeVector3(0, 0, -1) in OpenCV space) and normalizes the yaw difference into [-180, 180] degrees.
